# 跟我走，不會錯！🗾

福岡 7 日遊自由行行程 App，同時內建「🧳 行程本」——任何目的地都能建立新行程，AI 依當地季節、地形自動建議行程方向與備品清單。純網頁單檔案，部署在 GitHub Pages，支援手機加入主畫面、離線瀏覽。

🔗 線上網址（點我開啟）：[福岡7日遊.html](https://cyysongyy.github.io/fukuoka-trip/%E7%A6%8F%E5%B2%A17%E6%97%A5%E9%81%8A.html)

## 功能

- **📅 每日行程（福岡 2026/7/16–7/22）**：7 天行程卡片式呈現，交通、住宿、美食、注意事項一次看
  - 每張卡片有編號，可用 ▲▼ 按鈕自行調整當天卡片順序，一鍵還原原始順序
  - 每張卡片可新增自訂備註、可隱藏（軟刪除，隨時復原）
  - 每天可自行新增臨時項目（➕ 新增行程項目），適合旅途中臨時決定的行程
  - **📍 GPS 自動打卡**：開啟後靠近預設地標會自動標記「已到訪」
- **🍜 美食特輯 / 🎁 採買・伴手禮 / 🏪 超商必買**：分類整理的加碼景點與店家清單
- **🎒 備品清單**：出發前打包檢查表，可勾選、調整數量、自行新增/刪除品項，附進度條
- **🧳 行程本（萬能模板）**：不限福岡，任何旅程都能用
  - 輸入目的地、出發日期、天數、需求備註，AI 自動產生逐日行程草案＋備品建議
  - 備品建議會考慮當地當季氣候、地形，並給出一般性藥物類別提醒（不含劑量，僅供參考）
  - 產生後的行程、備品都能自行新增/編輯/刪除，也可重新請 AI 規劃
- **🤖 AI 旅遊小幫手**：串接使用者自己的 Gemini 或 OpenAI API Key，針對行程內容即時問答（行程本功能也共用這組設定）
- **🔒 私人資訊**：PIN 碼保護的自訂欄位（密碼、包車資訊等），僅存手機本機
- **🧭 我在哪**：一鍵開啟 Google 地圖目前位置
- **📤 備份／📥 還原**：一鍵把排序、備品、GPS 紀錄打包成文字，貼到備忘錄保存
- **PWA 離線支援**：加入主畫面後可離線瀏覽行程內容（AI 問答與行程本規劃需要網路）

## 檔案結構

```
福岡7日遊.html   主程式（單一 HTML 檔，含所有行程資料與邏輯）

sw.js            Service Worker，負責離線快取
icon-192.png     App 圖示 192x192
icon-512.png     App 圖示 512x512
```

## 使用方式

### 手機安裝（推薦）

1. 用 **Safari** 開啟上方網址（不要用 LINE 內建瀏覽器，會擋 App 部分功能）
2. 分享 → **加入主畫面**
3. 桌面會出現圖示，點開即為全螢幕 App 體驗
4. 首次開啟時請保持連網，讓 Service Worker 完整快取一次，之後即可離線瀏覽行程內容

### AI 小幫手 / 行程本設定

點底部「🤖 小幫手」，貼上自己的 API Key 即可使用（行程本的 AI 規劃也是用這組設定）：

- **Gemini**（免費）：[Google AI Studio](https://aistudio.google.com/apikey) → 用個人 Gmail 登入 → Create API key（選 in new project）
- **OpenAI**：[platform.openai.com](https://platform.openai.com/api-keys) → API keys（需綁定付費）

Key 只會存在使用者自己手機的 localStorage，不會上傳到任何伺服器。

## 資料與隱私

本 App 沒有後端伺服器，所有使用者輸入（備品清單、卡片排序、私人資訊、AI Key、行程本內容）都只存在**該支裝置的瀏覽器 localStorage**，不會同步、不會外流。清除瀏覽器資料或換裝置會導致這些自訂內容遺失（福岡行程本身的靜態內容不受影響）。行程本產生的內容會呼叫使用者自己設定的 AI API，內容僅供參考，特殊醫療/健康需求請自行諮詢醫師或藥師。
